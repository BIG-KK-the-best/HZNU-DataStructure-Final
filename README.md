# HZNU-DataStructure-final

本仓库内容是HZNU数据结构上机考的八个实验

## 主要内容概述

### graph

图论的两个实验，DFS和BFS
**实验中的图是用邻接表实现的**

### sort

排序算法的两个实验，**堆排序**和**快速排序**

### dataTransfer

数据转换，输入一个数，将其转换为二进制表示，要求用到**栈**，且链栈和顺序栈都要实现

### mergeList

有序列表的归并，要求使用链式存储或顺序存储的一种，这里我只实现了顺序存储。
